# Fortran Cmake Sample

## Setup

You must use cmake.

```shell script

$ brew install gcc

$ brew install cmake
```

## How to run

### First method

You have to install gcc(gfortran) and make, cmake for local environment.
```shell script
# コンパイルを行う
$　cd build && \
    cmake .. && \
    make

# コンパイル済みのコードの実行を行う
$ make run
```
